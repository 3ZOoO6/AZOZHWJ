## 🌟 المميزات

- دردشة موحدة بين عدة نماذج ذكاء
- اختيار البوت من قائمة bots (Gemini, DeepSeek, Blackbox)
- تصميم غامق جذاب واحترافي

- أكثر من 30 صفحة ثابتة
- تكامل مع Termux و Node.js و Flask
- صفحات HTML داكنة جاهزة للتوصيل
